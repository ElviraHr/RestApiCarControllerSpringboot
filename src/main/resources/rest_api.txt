GET получает данные GET/api/users /не меняет данные, безопасный инденпотентен
POST создает новый объект POST/api/user меняет данные, не безопасен, не иденпотентен
PUT обновление объекта PUT/api/users меняет данные не безопасен, иденпотентен
PATCH частичное обновление объекта, не нужно отправлять весь объект. Иденпотентен? и да и нет, зависит от ситуации
DELETE удаление объекта (по идентификатору) иденпотентен, действие необратимое, однократное

200 ОК все успешно
201 Created объект успешно создан GET
400 Bad request ошибка в данных
404 Not found объект не найден
500 Internal Server Error ошибка сервера
